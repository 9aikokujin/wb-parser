# Парсер товаров Wildberries

Парсер для сбора информации о товарах с сайта Wildberries по заданным критериям поиска.

## Описание

Скрипт собирает данные о товарах с Wildberries и сохраняет их в Excel-файл.
Поддерживает фильтрацию по цене, рейтингу и стране производства.

## Использование

1. Откройте `main.py` и при необходимости измените параметры в `CONFIG`:
```python
CONFIG = {
    'page': 1, # Первоначальная страница
    'min_price': 0, # Минимальная цена из фильтров
    'max_price': 10000, # Максимальная цена из фильтров
    'search_product': 'пальто из натуральной шерсти', # Поисковый запрос
    'min_rating': 4.5, # Минимальный рейтинг товара
    'RUS': 'f14177451=15000203' # Код производителя Россия
}
```
## Результат

Скрипт создает Excel-файл со следующими полями:
- Ссылка на товар
- Артикул
- Название
- Цена
- Описание
- Ссылки на изображения через запятую
- Характеристики с сохраненеим
- Название селлера
- Ссылка на селлера
- Размеры
- Остатки
- Рейтинг
- Количество отзывов

## Используемые библиотеки

- Python 3.8+
- Playwright
- pandas
- openpyxl
